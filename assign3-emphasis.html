  • to continue to become familiar with Photoshop and its various tools.

ASSIGNMENT

Please read the Emphasis Ebook before starting this assignment.

Using the principles of Unity and Emphasis learned thus far in Design and Communications you will create images that use type and symbols as basic shapes to visually communicates a point of emphasis in each composition.You will create 6 compositions with a definite focal point that use type as a design element beyond its role as what it signifies for language, and to create balanced images with rhythmic properties. When working with the 6 concepts, try the use the following techniques, at least once, to create emphasis:

  • isolation
  • contrast
  • placement

Begin by making preliminary sketches for the 6 concepts in the area indicated on the template. Use the Type tool to create letter and number forms. Fill each template box with letterforms, numbers, and symbols that maintain consistency in terms of style. Use the same font only in each box. You may use different fonts in each image. Work with different font sizes in each image to work with differences of scale. You may also utilize pattern backgrounds (of type) and gradients. You may use black, white and grayscale only.

Below are some ideas to keep in mind while working through the assignment:

  • Unity can be achieved by repetition, rhythm, similarity, or continuity.
  • Repetition can be achieved by use of similar fonts, repeating shapes, and/or alternating values.
  • Lack of Emphasis is created by reducing contrast and by repetitive or rhythmic placement - you want to avoid the lack of emphasis/or a focal point in your concepts.

SOFTWARE TIPS

Please note that these are only suggestions for your use, if you are familiar with the software and wish to use other tools, filters, gradients, etc. WITHIN THE PARAMETERS OF THE EXERCISE REQUIREMENTS, please feel free to do so. These tips are just to get you started or to help those still unfamiliar with the software.

For Assignment 3, you will use the Type tool and Transform commands, and the Move tool again in the layers. Create your new file, select the Type tool, and in the Options palette (on top of your screen directly below the title bar) select your font (experiment, find one that gives you the shapes and proportions you like), select it's style,(Roman, Italic, etc.), and type in 150pt for size.

Click in the image, and type one letter. Get the Move tool, and move the letter around your picture plane. Notice that it is a new layer with only the one letter on it. If you wish to turn the letter, go to Edit > Transform > Rotate. A small box will appear around your letter, and you can drag your mouse to rotate the letter at this time. When positioned at the angle you want, hit the Enter/Return key to set the transformation. You cannot go further until you either commit to the transformation or cancel it.

Repeat this for all the letters. Click on the correct layer to move them using the move tool, and with the layer active you can continue to rotate the letters individually using the above procedures.

ANALYSIS

Each week, you will be asked to respond to questions regarding your artwork created for each project. Please fill out the self-directed analysis, by answering the questions below when submitting your work in D2L.

Hint: sometimes D2L gets twitchy so I recommend writing your answers in your favorite text editor (Word, Google Docs, etc.) and then copy/paste into the D2L comment box.

Self-Directed Analysis:

  1. Explain how the element of unity and tehniques to create emphasis (contrast, placement, isolation) were used to create emphasis in the images.
  2. Explain which principles of the Gestalt theory have been used and where they were used in your concpets.
  3. Does the image meet the project objectives? Please explain how emphasis was created in each concept.
  4. What alternatives did you consider while doing this project?

Hint: These are terms that should be used in your analysis: Emphasis/ focal point, accents, containment, gestalt, repetition, scale, grid(s), unity/harmony.

EVALUATION

The project is worth 20 points.

The evaluation is based on:

  • ORGANIZATION / FORMAT: Imagery is organized in a clear, appropriate and precise manner. Images are unified by repetition, rhythm, similarity, and/or continuity. 6 points
  • EMPHASIS/FOCAL POINT: Imagery has a clearly defined focal point. An accent point supports the image. 11 points
  • SELF ASSESSMENT: Thoughtful evaluation of relevant issues correctly using the language of design. 3 points

Note the discussion forum posts and critiques are part of a separate assignment grade.
